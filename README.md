# Zippd Online Shopping DBMS

Zippd is an online shopping DBMS project built with Laravel and Microsoft SQL Server.

## What It Includes

- Product and category management
- Cart and checkout flow
- Orders, payments, and shipping tracking
- Reviews and returns workflow
- Admin dashboard queries and operations

## Tech Stack

- Laravel 11 (PHP 8.2+)
- Microsoft SQL Server
- Blade + Vite
- Docker Compose (local environment)

## Project Structure

- `laravel-app/` - Laravel web application
- `database/` - SQL schema, seed data, and query files
- `docs/` - ERD notes and supporting docs
- `docker/` - Docker image setup for the app
